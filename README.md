# 📦 Installation

Clone repository ini

```bash
git clone https://github.com/dukunhost11/repository.git
```

Masuk ke folder project

```bash
cd repository
```

Install dependencies

```bash
npm install
```

Jalankan project

```bash
npm run dev
```
